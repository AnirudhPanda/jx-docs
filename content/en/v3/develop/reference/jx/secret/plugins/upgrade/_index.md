## jx secret plugins upgrade

Upgrades the binary plugins of the secret command (e.g. the Vault binary)

### Usage

```
jx secret plugins upgrade
```

### Synopsis

Upgrades the binary plugins of the secret command (e.g. the vault binary)

### Examples

  ```bash
  # upgrades the plugin binaries
  jx upgrade

  ```
